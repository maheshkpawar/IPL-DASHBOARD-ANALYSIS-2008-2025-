# IPL Data Analysis (2008–2025) | Power BI Dashboard

## Project Overview
This project presents an interactive Power BI dashboard analyzing Indian Premier League (IPL) data from 2008 to 2025. The dashboard focuses on team performance, player achievements, boundary trends, and season-wise outcomes to deliver meaningful cricket insights for fans and analysts.

## Tool Used
- **Power BI** (Data Modeling, DAX, Interactive Dashboards)

## Dataset
- Historical IPL match and player data (2008–2024)
- IPL 2025 data is **simulated** and used purely for analytical and visualization practice

## Key Dashboard Features
- Season-wise slicer (2008–2025)
- Tournament summary (Total Teams, Matches, 4s, 6s)
- Orange Cap & Purple Cap analysis
- Top 5 highest individual scores
- Top 5 economical bowlers
- Boundary distribution (4s vs 6s)
- Points table with match results
- Winner and Runner-Up highlights

## Key Insights
- Boundary-heavy teams have a higher probability of winning matches
- Economy rate below 7 plays a crucial role in team success
- Consistent top-order batters dominate Orange Cap rankings
- Toss and venue factors influence match outcomes significantly
